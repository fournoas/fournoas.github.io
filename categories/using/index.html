Edge浏览器更新后，地址栏右侧会出现一个很恶心的“发现”按钮，在命令行中运行下面命令（或者直接快捷键 WIN+R 运行），然后重启浏览器就可以禁用这个按钮了：

REG ADD HKLM\Software\Policies\Microsoft\Edge /v HubsSidebarEnabled /t REG_DWORD /d 0
