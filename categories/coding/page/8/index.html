Lua 源码中定义了一个叫 L_Umaxalign 的数据类型：

ctypedef union { 
  double u; 
  void *s; 
  lua_Integer i; 
  long l; 
} L_Umaxalign;

这个结构其实是该平台上最长的数据类型。
