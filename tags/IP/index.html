从 APNIC 获取中国大陆地区的 IP 地址数据库

全球互联网 IP 地址分配由互联网号码分配局（IANA）机构负责。IANA 将分配工作委托给五大区域互联网机构管理（RIR），其中亚太网络信息中心（APNIC）负责管理亚太地区的事务。我们可以在 APNIC 的网站上找到所有分配给亚太地区的 IPv4/IPv6 地址数据库。通过该数据库，可以准确地知道某个 IP 地被分配给了……
