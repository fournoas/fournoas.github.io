编译 libmagic.dll

作者 Fournoas 写于 2025年2月14日 发布在 
  • Coding

libmagic 是由 file 命令提供的一个开源库，用于检测文件的类型，最新的版本号为 5.46，发布于 2024年 11 月。PyPI 上有一个名为 python-magic-bin 的包提供了编译好的 Windows 二进制版本，但最后一次更新已是 8 年前。Scoop 的 main 仓库中也包含了一个由第三方编译的 file 命令，版本号为 5.45，且没有提供 libmagic 的动态链接库。5.45 的 Magic 文件版本是 19，而 5.46 已经升级到 20，两者不兼容。

参考了 rkitover/file-windows 和 julian-r/file-windows 两个项目，我将原本的 Vistual Studio 编译器改成了 MinGW，项目发布在 GitHub：

tabris17/file-for-windows: file and libmagic for Windowsgithub.comfile and libmagic for Windows

可以在 Releases 页面下载编译好的二进制文件，也可以在本地下载。
