笔记软件 Cherrytree 试用体验

作者 Fournoas 于 2021年3月11日 发布在 
  • Using

Cherrytree 是一款支持多平台的笔记管理软件。软件支持树状分级和标签管理，数据以 XML 或 SQLite 格式保存。

主要特性：

  • 密码保护
  • 富文本和语法高亮
  • 可嵌入图片和文件
  • 导入纯文本、HTML、Markdown，以及其他笔记软件的文件格式
  • 导出纯文本、PDF、HTML
  • 按节点全文搜索

这些功能足够满足本人平时的笔记需求，然而在实际使用过程中，软件暴露出问很多问题。

首先，软件导入文件名不支持中文。这对于中文用户来说，导入功能简直就成了个鸡肋，完全没法用。

其次，软件的富文本编辑器非常难用，渲染出来的样式简直丑陋。从浏览器复制黏贴进来的HTML内容会丢失格式，布局错乱。如果你想直接将网页完整地保存到软件里几乎是不可能的。

另外，软件对 Markdown 的支持也非常有限。导入的 Markdown 文档经常会丢失内容，或者无法正确渲染。即便能渲染，样式也是丑到没法看。如果说编辑器难用这条还能忍受，那么这种丑陋是令人完全无法忍耐的。下图是 Cherrytree 和 Typora 对同一个 Markdown 文档的渲染效果对比：

还有，软件不支持远程访问，也没有移动端应用，无法在移动设备上访问笔记。在如今小屏使用多于大屏的时代，一款纯桌面版的笔记软件已经有点不合时宜了，应用场景非常有限。

最后，软件保存笔记的数据文件不支持增量备份，每次修改都需要上传完整的数据文件，不利于网盘进行数据同步。
