ReactOS

ReactOS 是一个开发与 Windows NT 和 Windows 2000 应用程序和硬件驱动程序兼容的开源操作系统的项目。

该项目从发起至今已有 20 余年，但由于种种原因，目前仍然处于 alpha 状态。

  • 官方网站：https://reactos.org

Haiku OS

Haiku 是一个开源软件项目，开始于2001年。目标是创立一个与BeOS 兼容的自由操作系统。2004年前，它称作 OpenBeOS。

Haiku 采用 C++ 语言开发，并提供面向对象的应用程序接口。Haiku 计划与 BeOS 在二进制与源代码上皆兼容。

  • 官方网站：https://www.haiku-os.org

Redox OS

Redox 是用 Rust 语言编写的类 Unix 微内核操作系统。Redox 是一个开源项目，参考了已有的内核和操作系统，包括 SeL4、MINIX、Plan 9 和 BSD。

  • 官方网站：https://www.redox-os.org

PureDarwin

Darwin 是由苹果公司于2000年所发布的一个开放源代码操作系统。Darwin 是 macOS 和 iOS 操作环境的操作系统部分。苹果公司于2000年把Darwin发布给开放源代码社群。

在 2002 年 4 月，Apple 成立 OpenDarwin 社区，创建它自己发布的 Darwin 操作系统。OpenDarwin 项目于2006年中止，2007 年，PureDarwin 项目成立去接手 OpenDarwin 之前的目标。

目前该项目仍处于 beta 阶段，离发布正式版遥遥无期。

  • 官方网站：http://www.puredarwin.org

FreeDOS

FreeDOS 是一个在 IBM PC 兼容机下运行的操作系统。FreeDOS 大部分兼容 MS-DOS，并允许一些早期版本的 Windows 像在 MS-DOS 上一样运行。

  • 官方网站：https://www.freedos.org

ArcaOS

ArcaOS 是一个基于 OS/2 的操作系统，它建立在 OS/2 Warp 4.52 的基础上，增加了对新硬件的支持，修复了操作系统中的缺陷和限制，并加入了新的应用程序和工具。

  • 官方网站：https://www.arcanoae.com/arcaos/

AmigaOS

AmigaOS 最开始于 1985 年由 Commodore International 公司开发，适用于 AmigaONE PC 的个人操作系统。

自 AmigaOS 4 开始，操作系统运行于 PowerPC 处理器上。

  • 官方网站：https://www.amigaos.net

AROS

AROS 是一款类 AmigaOS 的开源操作系统。项目始于 1995 年，多年来已成为 AmigaOS 的几乎功能完整的实现，是 AmigaOS 的完整免费开源替代品。

AROS 支持 x86 和 PowerPC 架构的个人计算机。

  • 官方网站：http://www.aros.org

RISC OS

由 Acorn 公司（ARM 公司的前身）开发，最初用于该公司于 1987 年发布的 Archimedes 个人电脑。最新版本可运行在树莓派上。2018年10月，RISC OS 在Apache 2.0 许可证下开源。

  • 官方网站：http://riscosopen.org

MenuetOS

MenuetOS 由芬兰人 Ville Mikael Turjanmaan 开发的操作系统。它由汇编语言写成，安装镜像文件仅 1.44 MB 大小。Menuet OS 的 32 位版本 Menuet32 使用 GNU 通用公共许可证发放，但 64 位版本 Menuet64 使用自己的协议发放。

  • 官方网站：http://menuetos.net

KolibriOS

KolibriOS 是一个很小但非常强大和快速的操作系统，只需要几兆的磁盘空间和 8 MB 的内存即可运行。

该项目于 2004 年从 MenuetOS 分叉出来，所有的代码都是开源的，大部分代码都是在 GPLv2 许可证下发布的。

  • 官方网站：https://www.kolibrios.org

ToaruOS

ToaruOS 是一个由 K. Lange 开发的类 Unix 操作系统。名字的出处是动画《某科学的超电磁炮》（Toaru Kagaku no Rērugan）

  • 官方网站：https://toaruos.org

HelenOS

HelenOS 是一款从零开发的、基于微内核的、可移植的 multiserver 操作系统。HelenOS 可在八种不同的处理器架构和机器上运行，从嵌入式 ARM 设备、单片机到多核 32 位和 64 位台式电脑，再到 64 位安腾和 SPARC 机架式服务器。

HelenOS是开源的免费软件。其源代码在BSD许可证下可用。某些第三方组件是根据 GPL 授权的。

  • 官方网站：http://www.helenos.org

SerenityOS

SerenityOS 是一个免费的开源类 Unix 桌面操作系统，自2018年以来一直在持续开发。SerenityOS 最初是瑞典程序员 Andreas Kling 的个人项目，现在由业余爱好者社区开发。该系统支持 x86 和 x86-64 指令集，具有抢占式内核，并托管多个复杂的应用程序，包括自己的 Web 浏览器和集成开发环境。

  • 官方网站：https://www.serenityos.org

基于 BSD

GhostBSD

简单、优雅的桌面 BSD 操作系统。其默认的桌面环境是 MATE。

  • 官方网站：http://www.ghostbsd.org

TrueOS

曾用名 PC-BSD，是历史最悠久的桌面 BSD 操作系统。可惜已经于 2020 年停止开发了。原开发团队转而开发基于 BSD 的 TrueNAS 操作系统。

  • 官方网站：https://trueos.org

NomadBSD

适用于 USB 闪存驱动器的 Live 系统。与自动硬件检测和设置一起，它被配置为一个开箱即用的桌面系统，但也可以用于数据恢复、教学或测试 FreeBSD 的硬件兼容性。

  • 官方网站：https://www.nomadbsd.org

helloSystem

用户界面模仿 macOS 的 BSD 操作系统。项目还处于实验性阶段。

  • 官方网站：https://hellosystem.github.io

MidnightBSD

MidnightBSD 拥有一个由专门的开发人员组成的小型社区，致力于创建一个每个人都可以自由使用的易于使用的操作系统。适用于 x86、AMD64 和虚拟机。

  • 官方网站：https://www.midnightbsd.org

ravynOS

曾用名 airyxOS。除了用户界面模仿 macOS 外，普通的 macOS 和 Darwin 应用程序可以使用开发中的转换器直接在 ravynOS 上运行。基于 AppKit 的源代码可以在本机生成和运行。

  • 官方网站：https://www.ravynos.com

Linux 发行版

elementary OS

elementary OS 是一个基于 Ubuntu 的发行版。官方宣传口号是：深虑、全能、有品德的 Windows / macOS 替代品。

  • 官方网站：https://elementary.io

Linux Mint

Linux Mint 是一个基于 Ubuntu 的发行版。它是台式机和笔记本电脑的操作系统，旨在开箱即用地工作，并配备了大多数人需要的应用程序。

  • 官方网站：https://linuxmint.com

PureOS

PureOS 是使用 GNOME 桌面环境的 Linux 发行版，主要关注隐私和安全性。PureOS 旨在仅包含免费/自由和开源软件 (FOSS/FLOSS)，并且包含在自由软件基金会发布的免费 Linux 发行版列表中。

  • 官方网站：https://pureos.net

Asahi Linux

专为 Apple Silicon 硬件平台定制的 Linux 发行版。

  • 官方网站：https://asahilinux.org

Qubes OS

Qubes OS 是一个专注于安全的桌面操作系统，旨在通过隔离提供安全保障。通过Xen来执行虚拟化，用户环境可以基于 Fedora、Debian、Whonix 和 Microsoft Windows 或其他操作系统。

  • 官方网站：https://www.qubes-os.org

Zorin OS

Zorin OS 是一个基于 Ubuntu 的 Linux 发行版。其目的是尽可能的模拟 Windows 的操作习惯，让用户可以方便的使用。为了让 Windows 用户更方便的使用，Zorin OS 预装了 Wine。Pro 版可以将用户界面切换成 macOS 和 Windows 风格。

  • 官方网站：https://zorin.com

Nitrux

Nitrux 是基于 Debian 的 Linux 发行版。使用 KDE Plasma、OpenRC 和 AppImage。内核使用 XanMod Kernel。

  • 官方网站：https://nxos.org
