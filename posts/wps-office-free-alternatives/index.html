替代 WPS Office 的免费软件

作者 Fournoas 于 2022年7月11日 发布在 
  • Using

听说最近 WPS 又翻车了。软件被指审查用户文档内容，可导致文档被锁定无法打开。

也有人指出，这是因为用户开启了云端同步，文档上传后触发服务器敏感词检测所导致的。

但无论如何，WPS Office 已经被人诟病已久。比如文档莫名其妙地损坏，必须缴费成为会员才能恢复；还有像狗皮膏药一样无法删除的 WPS 云服务。

特此推荐几个能够替代 WPS 的免费软件。

WPS Office 国际版

由于国内法律法规制定和执行的落后，国产软件通常会区别对待用户，把国内用户视作成劣等贱民。这些软件的中文版往往垃圾又恶心，而国际版却是干净又卫生。比如 QQ，又比如 WPS。

虽然 WPS Office 国际版也有讨厌的云服务，但软件总体来说还算是规规矩矩的。唯一的遗憾是软件界面不能选择中文。理论上可以自制汉化包，并且本人在旧版本的 WPS 国际版上成功实现汉化，但是最新版本的 WPS 能否汉化还不得而知。

  • 官方网站：https://www.wps.com
  • 支持平台：Windows / Linux / macOS

Google Docs

Google 的文档在线协作处理平台。无需安装任何软件，只要浏览器打开即可使用。缺点是需要科学上网才能访问。

同类的文档在线写作平台很多，但是个人认为 Google Docs 是最好用的一款。其他在线平台不再推荐。

  • 官方网站：https://www.google.com/docs/about/

LibreOffice

LibreOffice 最初是 OpenOffice 的分支，但是现在的社区活跃程度远远高于 OpenOffice。LibreOffice 采用 OpenDocument 作为默认的文档格式，同时也支持 MS Office 格式，不过兼容性并不是很好。LibreOffice 也是本文介绍的软件中，支持格式最多的一款。

LibreOffice 对 Windows 平台的支持也相当完善。不过 LibreOffice 在 Windows 上编译使用了 Cygwin，性能略有损失，在低配硬件上运行可能不太流畅。

  • 官方网站：https://www.libreoffice.org
  • 支持平台：Windows / Linux / FreeBSD / macOS

NeoOffice

NeoOffice 是运行于 macOS 平台上，基于 LibreOffice 和 OpenOffice 开发的办公套件。在两款软件的基础上又增加了一些功能，特别是支持 iCloud、DropBox 等云盘。

  • 官方网站：https://www.neooffice.org
  • 支持平台：macOS

FreeOffice

FreeOffice 是 SoftMaker Office 的免费版。支持 MS Office 和 ODT 文档格式，还支持 PDF 格式导出。软件界面上接近于传统的 MS Office。对于普通用户，免费的 FreeOffice 的功能已经足够应付日常使用了。

  • 官方网站：https://www.freeoffice.com
  • 支持平台：Windows / Linux / macOS

ONLYOFFICE

ONLYOFFICE 是一个开源项目。其中 ONLYOFFICE Desktop Editors 可以运行在 Windows、Linux 和 macOS 上。支持 OpenDocument、epub、pdf 以及 Office Open XML 格式（.docx, .xlsx, .pptx）。

ONLYOFFICE Desktop Editors 基于 Chromium Embedded Framework 开发，本质上就是一个内嵌了 Chrome 浏览器内核的网页应用，所以性能不及原生应用。

此外，软件支持 ownCloud、Seafile、Nextcloud、kDrive、Liferay 和 ONLYOFFICE 云存储。

  • 官方网站：https://www.onlyoffice.com
  • 支持平台：Windows / Linux / macOS

Calligra

Calligra Suite 是 KDE 的办公软件项目。软件采用 OpenDocument 作为默认文档格式，对 MS Word 的格式支持很差。而在 Windows 平台上，官方提供的是旧版的 Calligra Gemini，用起来非常糟心，且默认没有提供中文界面。另外，在本人测试过程中，经常碰到闪退的现象。因此，不推荐 Windows 用户使用该软件，Linux 用户或许可以试一试。

  • 官方网站：https://calligra.org
  • 支持平台：Windows / Linux / FreeBSD / macOS
