安装纯净版腾讯 QQ

作者 Fournoas 于 2020年6月24日 发布在 
  • Using

众所周知，腾讯 QQ PC 版是个流氓软件，会在系统种塞入各种垃圾进程，比如 QQProtect.exe。这些进程不但拖垮系统性能，并且由于运行在系统权限下，给系统安全和用户隐私带来各种问题。而 QQ 国际版已经多年未更新了，很多新功能不被支持。

网上有第三方修改的绿色版 QQ 安装包，不过个人不建议使用。对于 Win 10 用户，可以从微软应用商店里安装 QQ。如果是旧版本的 Windows 或者系统内没有微软应用商店，则可以通过以下方法，实现手动安装。

  1. 打开 https://store.rg-adguard.net/
  2. 输入 https://www.microsoft.com/zh-cn/p/qq%E6%A1%8C%E9%9D%A2%E7%89%88/9nhlgf0zwc5s
  3. 下载红圈标注的 appx 安装包
  4. 用解压缩软件打开该 .appx 文件
  5. 提取安装包中的 QQ 文件夹
  6. 运行 QQ\bin\QQ.exe 启动 QQ
