如何修复 Failed to load JVM DLL 错误

作者 Fournoas 于 2022年6月11日 发布在
  • Using

在一台全新的 Windows 10 21H2(19044.1706) 系统上安装了 IntelliJ IDEA。启动 IDEA 时报错，错误信息为：Failed to load JVM DLL。

IDEA 和 JDK 都是通过 scoop 安装的，确信安装没有问题。环境变量 JAVA_HOME 和 JDK_HOME 都设置正确。通过 Stackoverflow 检索发现，有人通过 Run As Administrator 可以解决此问题。遂测试之，发现成功。虽然非常奇怪：以前无论是通过 scoop 安装还是手动安装，都从未碰到过这个问题。
