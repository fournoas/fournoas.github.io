配置音乐播放器 foobar2000

作者 Fournoas 写于 2022年1月27日 发布在 
  • Using

foobar2000 是一款老牌的免费音乐播放器，作者是 Winamp 开发组 Nullsoft 的成员。它的特点是界面简洁（陋），并提供丰富的解码器和插件，可配置性高。也正因为如此，这款播放器在没有配置的情况下用起来并不趁手。虽然网上有各种爱好者发布的开箱即用的打包版本，不过出于安全性考虑，不建议使用这些第三方发布的版本。

安装播放器

官网下载地址：https://www.foobar2000.org/download，当前最新版本是 1.6.9。

默认安装的 foobar2000 只提供 Default UI，要支持复杂的用户界面，需要安装 Columns UI 插件。下文会给出安装和配置方法。

曾经还有个 Panels UI 的用户界面插件，可以用来美化用户界面，但现在已经不再支持。

软件第一次运行时会弹出 “Quick Appearance Setup” 对话框用于设置 Default UI。因为之后要切换到 Columns UI，所以此处无需设置，点击 Cancel 即可。

安装插件

本文推荐安装的插件有：

  1. Columns UI 用户界面
  2. OpenLyrics 歌词面板
  3. Album list panel 专辑列表面板
  4. Monkey’s Audio Decoder APE 解码器
  5. Youtube Source 播放 Youtube 视频

下载插件文件后，双击打开即可安装。安装后需要重启播放器。某些插件页面没有提供直接下载地址，可以访问 Home page 下载。

安装完 Columns UI 插件后，运行软件会弹出 “New user interface module(s) found” 对话框，选择 Columns UI，点击 OK。之后弹出 “Quick Setup - Columns UI” 对话框，点击 Cancel，之后我们将会手动设置。

设置用户界面

在软件主界面按下 Ctrl+P，打开 Preferences 配置对话框。展开 Display - Columns UI - Layout 选项页。在右侧树形列表中的项目上按下右键，点击右键弹出菜单中的 Remove panel，逐一删除列表中的项目，直至剩下唯一的 Horizontal splitter。

然后在列表上按下右键，在 Insert panel 子菜单中选择需要添加的项目。按照下图所示逐个添加项目。

特别地，第一个和第三个 Vertical splitter、Item properties、Artwork view 项目要勾选右侧的 Locked 选项，其余选项都保持未勾选状态。这样播放器器窗口缩放时，这些面板能保持尺寸。

设置插件

大多数插件并不需要特别设置即可使用。你可以在界面上右键里选择 Settings、Options、Preferences 等菜单中对这些面板进行简单的设置。

其中 Youtube Source 插件需要安装额外的程序才能正常打开视频，如果你仅仅是用来收听 Youtube 音频的，那就无需做以下的操作。

打开 Preferences 配置对话框，展开「Youtube 来源」选项页，切换到「维护」选项卡，在「第三方二进制文件」列表中补全缺少的程序。

在我的系统中，需要安装 LAV Filters 和 madVR 两个程序。如果需要支持字幕，还需要安装 VSFilter。可以手动下载安装，也可以使用 Scoop 安装：

powershellscoop install lavfilters madvr

配置完成后，在播放 Youtube 视频时，可以选择菜单 View - Virtualizations - 「影片」来打开视频窗口。不过 Youtube source 播放视频并不怎么流畅，所以还是当作音频源收听比较合适。
