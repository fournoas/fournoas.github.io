浏览器

  • Google Chrome
    谷歌浏览器。
  • Vivaldi
    Opera 浏览器被奇虎 360 收购后，其公司创始人另起炉灶创办了 Vivaldi Technologies 公司。Vivaldi 基于浏览器 Google 的 Blink 引擎。
  • Microsoft Edge
    微软的新版浏览器。基于 Google 的 Blink 引擎。
  • SeaMonkey
    基于 Mozilla Suite 的互联网软件套装。软件包含浏览器、邮件和新闻组客户端、HTML 编辑器、IRC 聊天客户端。
  • Brave
    一款注重保护用户隐私的浏览器，内置 Tor 功能。基于 Google 的 Blink 引擎。

网络

  • Free Download Manager
    网络下载工具。可视作免费版的 IDM。
  • FileZilla
    免费 FTP 工具。包括客户端和服务器端口两款软件。另外还有收费的 Pro 版本，支持更多协议和各种主流云盘。
  • Aria2
    支持多种协议的命令行下载工具。
  • WinSCP
    使用的 SSH 的开源图形化 SFTP 客户端。同时支持 SCP 协议。
  • Charles *
    网页抓包工具。免费试用 30 天。
  • Fiddler
    免费版的网页抓包工具。
  • Wireshark
    网络嗅探/抓包工具。需要 WinPcap 驱动支持，Win 10 系统可以使用 Npcap 或 Win10Pcap 替代。
  • MobaXterm
    一款 Windows 版的 X Server。支持 SSH、Telnet、VNC、RDP等协议。
  • Privoxy
    一款不进行网页缓存且自带过滤功能的代理服务器。
  • PuTTY
    Windows 下的 SSH 客户端。
  • FreeRDP
    一款开源的 RDP 客户端。
  • I2P
    分布式匿名网络。
  • ZeroNet
    基于Bitcoin加密和BitTorrent技术构建的分布式网络。
  • Resilio Sync
    基于P2P技术的跨平台数据同步工具。除 Windows 外，还支持 macOS、Linux、Android、iOS、FreeBSD。
  • SoftEther VPN
    开源跨平台、支持多种协议的 VPN 软件。
  • TightVNC
    使用 VNC 协议的远程桌面服务器端和客户端。支持 Windows、Linux/Unix 和 macOS。
  • Miranda NG
    老牌多协议即时通讯软件 Miranda IM 的后续版本。支持 ICQ, IRC, Jabber 等多种通信协议。
  • HTTP Downloader
    使用 IOCP 实现的高性能下载工具。支持 HTTP(S)/FTP(S)/SFTP 协议。
  • WireGuard
    一款开源、跨平台的 VPN 软件，旨在获得比 IPsec 和 OpenVPN 更好的性能。该项目还为 Windows 平台提供了一款简单小巧的 TUN 驱动——Wintun。
  • Clash For Windows
    代理工具 Clash 的 GUI。
  • mitmproxy
    一款开源交互式的 HTTPS 代理。
  • HTTP File Server
    个人文件共享服务器。可以将本地文件以 Web UI 的形式共享，并且可以设置用户权限。
  • CuteHttpFileServer
    CuteHttpFileServer/chfs 是一个跨平台 HTTP 协议的文件共享服务器，使用浏览器可以快速访问。支持 HTTPS 和 WebDAV。
  • dnscrypt-proxy
    在本地开启 DNS 加密代理服务。

桌面辅助

  • Fences *
    桌面图标整理工具。
  • Rainmeter
    桌面自定义工具。
  • Coodesker
    酷呆桌面。自定义桌面和图标整理工具。
  • CairoShell
    Windows Shell 替代软件。
  • QuickLook
    文件快速预览工具。
  • Lingoes
    桌面词典工具。
  • Lightshot
    截图工具。自带图床，非常小巧，安装后仅 4 M。功能比较简单。
  • Snipaste
    截图工具。软件自带简单的图片处理功能。
  • ShrareX
    开源截图与录屏软件，支持滚屏截图，GIF 屏幕录制，上传图床。
  • uTools
    跨平台桌面工具集成软件，支持插件。基于 Electron.js 开发，占用资源且性能存在问题。
  • Ditto
    剪贴板增强工具。可以把复制过的所有内容保存起来。
  • Nimi Places
    桌面图标整理工具。

系统工具

  • 7-Zip
    足够简单、小巧、够用的压缩软件。
  • Bandzip
    Bandizip 是一个强大的多功能压缩文件管理工具，可提供高速的解压缩速度和其他各种实用功能。
  • Chocolatey
    Windows 下的软件包管理器。有图形界面。
  • Scoop
    Windows 下的命令行包管理器。
  • LockHunter
    文件解锁工具。可以查找哪个进程占用了当前文件。安装后集成到资源管理器的右键菜单。
  • Unlocker
    LockHunter 的同类软件。已经停止开发，最新版本是 1.9.2，官方仅支持到 Win 7 系统。
  • SpaceSniffer
    以图表直观显示磁盘的使用情况。
  • ConEmu
    支持选项卡的终端。可以替代 Windows 的 cmd 命令行。
  • Registry Workshop *
    高级注册表编辑器。用来搜索键值非常快速。
  • Geek Uninstaller
    软件卸载工具。可以彻底删除软件残留的文件。
  • CCleaner
    系统清理软件。
  • Sysinternals Suite
    Windows 系统工具套装。其中几个有用的工具：
    • Process Monitor
      监控进程对注册表、文件系统、网络和进程/线程的访问活动。
    • Desktops
      创建多个桌面，并通过快捷键切换。
    • Autoruns
      查看系统自启动项目。
    • Process Explorer
      查看进程信息。
  • VirtualBox
    虚拟机软件。
  • Genymotion
    一款 Android 模拟器。
  • VeraCrypt
    一款跨平台的磁盘加密软件，支持 Windows、Mac OS、Linux、FreeBSD。是 TrueCrypt 的替代软件。
  • Cygwin
    运行于 Windows 上的 Linux 的命令集。并提供了一套 POSIX 接口。
  • MSYS2
    类似 Cygwin 的 Linux 工具集。和 Cygwin 相比更加小巧，安装后文件所占大小是 Cygwin 的 1/10 左右。
  • BusyBox
    运行于 Windows 上的 UNIX 工具集和功能有限的 Bash 解释器。相比 Cygwin 更加小巧。
  • CPU-Z
    系统硬件信息查看工具。
  • Eraser
    安全删除文件工具。文件删除后保证无法恢复。
  • Sandboxie
    沙盒程序。2019年9月11日，Sandboxie转型成免费软件，并于2020年3月25日以GPL V3许可协议开放源代码，正式成为开源软件。
  • PowerToys
    微软官方推出的 Windows 实用工具套装。现已经开源。仅支持 Windows 10 v1803 以上版本。
  • WinGet
    微软官方推出的 Windows 软件包管理器。仅支持 Windows 10 v1709 以上版本。
  • PortableApps
    绿色软件平台。
  • IObit
    • Advanced SystemCare *
      系统优化和清理软件。有免费版和收费版。
    • IObit Uninstaller *
      软件卸载工具。有免费版和收费版。
    • IObit Unlocker
      文件解锁工具。
    • IObit Undelete
      删除文件恢复工具。
  • WinImage *
    镜像制作工具。共享软件，可免费试用30天。
  • IsoCreator
    可创建 ISO 9660 格式的 CD 镜像。
  • Daemon Tools *
    虚拟光驱软件。Lite 版可免费使用（带广告）。
  • Alcohol 120% *
    虚拟光驱和光盘刻录软件。15 天免费试用，免费版限制只能使用 2 个虚拟光驱和同时只能写入 1 个光驱。
  • NirSoft Utilities
    • ShellExView
      系统右键菜单管理工具。
    • FileTypesMan
      文件类型管理工具。
  • O&O
    • ShutUp10++
      系统设置工具。
  • Windows System Control Center
    集成了 Sysinternals Suite、NirSoft Utilities、MiTeC Utilities 等 Windows 系统工具。
  • Rufus
    创建 USB 引导盘的工具。
  • Windows File Recovery
    微软提供的用于恢复已删除文件的工具。
  • DISM++
    DISM 命令的 GUI 版本，且额外具有电脑管家功能。除了系统清理优化功能外，还可以用来删除系统预装的 Appx 应用。

输入法

  • 小狼毫输入法
    完全脱机运行的输入法，保护个人隐私。缺点是比较难用。
  • Google 拼音输入法
    已经停止开发很多年了，不过仍然可以使用。优点是该输入法的在线服务已经下线，输入法再也不会泄露你的个人隐私了。

文件管理

  • FreeCommander
    免费版的 Total Commander。当然功能上要差很多。
  • Explorer++
    支持标签页浏览的文件管理器。
  • Double Commander
    开源跨平台文件管理器，支持双窗口标签页。界面有点丑。
  • One Commander
    支持双窗口标签页的文件管理器。界面风格比较现代。无需配置即可支持 QuickLook。个人用户可以免费使用，商业用户需要注册。
  • Directory Opus *
    一款类似 TC 的文件管理软件。价格比 TC 便宜一点。
  • Q-Dir
    默认四个文件列表窗口的文件管理器。支持多种布局。
  • Files App
    基于 UWP 的现代文件管理器。支持多标签。和 Windows 自带的资源管理器比，除了界面美观外，并没有太大优势。
  • Everything
    快速文件索引工具。通过文件名快速查找文件。
  • AnyTXT Searcher
    文本文件全文检索工具。
  • FastCopy
    文件批量复制工具。
  • DSynchronize
    文件同步工具。

媒体播放器

  • VLC media player
    界面简洁易用的媒体播放器。
  • SMPlayer
    支持 Windows 和 Linux 平台的媒体播放器。使用 MPlayer 或 mpv 作为播放引擎。
  • Media Player Classic - Black Edition
    自 MPC-HC 在 2017 年停止开发后，这款是 Windows 经典播放器 MPC 仅存的继承者了。
  • Foobar2000
    经典的音乐播放器。
  • AIMP
    一款来自俄罗斯的音乐播放器。中文用户记得在选项-界面中的缩放设置为 125%。
  • MusicBee
    MusicBee是使用BASS音频库构建的免费媒体播放器，用于在Microsoft Windows上播放和组织音频文件。
  • K-Lite Codec Pack
    音频及视频格式的解码器安装包。
  • Audacious
    开源跨平台音乐播放器。界面类似 Foobar2000。
  • DeaDBeef
    同样是一款界面朴素的开源跨平台音乐播放器。

影音图像处理

  • GIMP
    一款历史悠久功能强大的开源图像编辑软件。
  • XnView
    图片浏览器。可以进行简单的图像编辑操作。该软件分别有 XnView MP 和 XnView Classic 两种版本，前者是跨平台版本，后者是 Windows 平台专用版本。后者的打开速度更快些。
  • Honeyview
    韩国公司开发的一款快速的图片查看器。
  • Honeycam
    同一家韩国公司开发的GIF/WebP动图制作和编辑器。可以录制屏幕。
  • MangaMeeya
    最好用的漫画阅读器。可惜已经停止开发，只能在第三方下载网站上找到踪迹。
  • IrfanView
    小巧但并不太好用的看图软件。
  • FastStone Image Viewer
    一款小巧的图片浏览器。家庭用户可以免费使用，商业用户需要购买许可。
  • Krita
    一款开源的绘图软件。
  • Hitfilm Express
    一款免费视频编辑软件。另外有收费的 Hitfilm Pro 版。
  • Paint.NET
    一款开源的图像编辑软件。和 GIMP 相比功能比较弱。
  • PhotoDemon
    免费、免安装、开源、小巧的图像编辑软件。
  • OpenShot
    一款开源的视频编辑软件。
  • Shotcut
    一款开源的视频编辑软件。
  • FFmpeg
    一款开源的音视频处理工具。Windows 下有 GUI 工具 WinFF。
  • LAME
    一款开源的 MP3 编码工具。Windows 下有 GUI 工具 winLAME。
  • OBS
    视频直播推流工具。
  • SpleeterGUI
    可以将音频中的人声和各种乐器的声音分离。使用音乐网站 Deezer 数据进行训练。
  • Demucs GUI
    和 Spleeter 类似的工具。使用 MusDB HQ 数据进行训练。

设计

  • SketchUp
    最新的 SketchUp 免费版已经不再提供离线安装版本，而仅有在线编辑版本。网上有 SketchUp 2017 安装包下载。
  • Blender
    一款开源的 3D 建模软件。
  • FreeCAD
    一款开源的 CAD 软件。
  • QCAD
    跨平台 2D-CAD 软件。有开源的社区版和收费的商业版。
  • LibreCAD
    跨平台开源 2D-CAD 软件。是 QCAD 社区版的分支。
  • SolidWorks *
    三维机械 CAD 软件。学生可以购买 Student Edition，价格十分便宜。
  • Sweet Home 3D
    室内装修设计软件。
  • Scribus
    一款开源的桌面排版软件。
  • Dia Diagram Editor
    一款开源的流程图绘制软件。
  • Pencil Project
    开源的流程图和原型设计工具。

文档处理

  • SumatraPDF
    一款小巧的 PDF 阅读器。软件启动和文档打开速度极快。
  • XpdfReader
    开源跨平台的 PDF 阅读器。该项目还提供一套 xpdf-tools 格式转换工具。
  • Typora
    一款 Markdown 编辑器。
  • Pandoc
    文档格式转换工具。
  • PDF Creator
    通过虚拟打印机创建 PDF 文档。
  • CutePDF Writer
    一款类似 PDFCreator 的 PDF 生成软件。
  • Tesseract OCR
    Google 开发的一款开源 OCR 软件。
  • wkhtmltox
    可以将 HTML 转换成 PDF 或图片的工具。
  • 福昕PDF编辑器
    福昕的 PDF 阅读器越做越垮。不过福昕的PDF编辑器还是可以用用。个人版可以免费使用，但是功能受限。
  • WinMerge
    文件比较、合并工具。
  • TeXstudio + MiKTeX
    LaTeX 编辑器和编译器。
  • Xournal++
    开源跨平台的笔记软件。支持压敏笔绘图板手写输入，内置 LaTeX 编辑器，可导出 PDF 文档，可以为 PDF 做注释。
  • Office Viewer
    VSCode 的插件。WYSIWYG 的 Markdown 编辑器，并且支持查看 Excel 文档（.xls, .xlsx, .csv）、PDF 文档、字体文件（.ttf, .otf, .woff），SVG 矢量图等。
  • calibre
    电子书管理软件。除了阅读和管理电子书外，还支持格式转换、制作和编辑电子书。支持几乎所有常见的电子书格式，包括 EPUB、MOBI、AZW3、DOCX、FB2、HTMLZ、LIT、LRF、PDB、PDF、PMIZ、RB、RTF、SNB、TCR、TXT、TXTZ 等。缺点是打开文件速度比较慢，界面古板。
  • FreePDF
    免费的 PDF 编辑器。

十六进制编辑器

  • UltraEdit *
    老牌文本编辑器，提供十六进制编辑功能。就是价格太贵了。
  • WinHex *
    许可证按年订阅，个人版一年300元人民币，有点小贵。
  • Hex Workshop *
    非常好用的十六进制编辑器，内置常见文件格式的数据结构模板。单一许可证600元人民币。
  • Free Hex Editor Neo
    一款免费的十六进制编辑器。不怎么好用，很多功能只有收费版才提供。
  • HxD
    在免费的十六进制编辑器中比较好用的一款。
  • wxMEdit
    基于已经停止开发的开源文本编辑器 MadEdit 改进的十六进制编辑器。功能相对单一。
  • wxHexEditor
    同样功能相对单一。
  • ImHex
    功能比较丰富的开源十六进制编辑器。

文本编辑器

  • Notepad++
    支持多页切换的文本编辑器。
  • Notepad3
    单页文本编辑器。可以用来替换系统自带的记事本。
  • Geany
    基于 Scintilla 编辑控件开发的跨平台文本编辑器，可以作为轻量级 IDE 使用。
  • CudaText
    使用 Lazarus 开发的跨平台文本编辑器。在 Windows 平台上显示汉字有文字重叠的 BUG。

办公

  • CherryTree
    支持多平台的笔记软件。支持自由节点，自定义标签。
  • Obsidian
    笔记软件，可以作为本地化的知识库。支持构建知识图谱。
  • LibreOffice
    开源 Office 系统。可以兼容 Microsoft Office 文件格式。
  • WPS Office
    建议安装英文版，或者通过微软应用商店安装。
  • Mailbird *
    邮件客户端。还集成了任务管理、文件共享、视频会议等功能。Lite 版本可以免费使用，仅支持一个账号。
  • Claws Mail
    开源的跨平台邮件客户端和新闻聚合器。2005 年从 Sylpheed 分叉出来的项目，现在相互独立开发。
  • FreeOffice
    SoftMaker Office 的免费版本。支持 Windows、Linux 和 macOS。免费版功能不全，但是对普通用户来说足够了。

娱乐

  • Kodi
    家庭影院服务。
  • Plex Media Server
    家庭影院服务。
  • Emby
    家庭影院服务。

编程

  • Cppcheck
  • Visual Studio Code
  • Postman
  • PE Explorer *
  • Poedit
  • Source Insight *
  • Vagrant
  • Windows Subsystem for Linux (WSL)
  • HeidiSQL
  • DBeaver
  • Another Redis Desktop Manager
  • Visual C++ Build Tools
  • IDEA Community Edition
  • CodeBlocks
  • PyCharm *
  • MinGW / MinGW-w64
  • IDA Pro *
  • OllyDbg
  • Dependency Walk
  • LLVM
  • Lazarus
  • Ghidra
  • SourceTree
  • DB Browser for SQLite
  • Robo 3T
  • MonoDevelop
