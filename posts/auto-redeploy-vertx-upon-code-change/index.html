Vert.x 项目自动重新部署

作者 Fournoas 于 2020年4月24日 发布在 
  • Coding

Vert.x 是一款基于 JVM 的高性能异步框架。使用官网提供的项目生成器创建的 gradle 配置有一个 BUG，导致在 Windows 环境下无法完成自动部署。

修复方法如下：

打开项目根目录下的build.gradle文件。找到

SCALAdef doOnChange = './gradlew classes'

修改成

SCALAdef doOnChange = ".${File.separator}gradlew classes"

即可。

问题的原因是 Windows 无法识别 Linux 的/路径分隔符，导致运行gradlew classes命令失败。
